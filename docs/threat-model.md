# Модель угроз (кратко), Shectory Assist MVP

## Активы

- Токен Telegram-бота (`TELEGRAM_BOT_TOKEN`) — полный контроль над ботом.
- Ключ Gemini (`AGENT_LLM_API_KEY`) — квоты и доступ к API.
- Содержимое чатов (транскрипты, тексты).
- При webhook: `TELEGRAM_WEBHOOK_SECRET` и URL эндпоинта.

## Поверхностные угрозы и меры

| Угроза | Мера в MVP |
|--------|------------|
| Утечка секретов из репозитория | Только `.env` / секрет-хранилище; `.env` в `.gitignore`. |
| Подделка webhook | Проверка `X-Telegram-Bot-Api-Secret-Token` при заданном `TELEGRAM_WEBHOOK_SECRET`. |
| Злоупотребление ботом | Rate limit на пользователя в оркестраторе; лимит запросов к gazeta.ru в навыке. |
| Утечка ПДн в логах | Не логировать сырое аудио; текст — усечённо. |

Парсинг главной Gazeta.ru — юридический риск; для продуктивной версии — RSS/API (см. ТЗ и ADR).
